## 前言

> 本工程是一个基于java实现的爬虫工具包，初衷是想能简单高效爬取互联网数据，我个人比较感兴趣的还是股票数据，因此测试用例全是爬的股票数据
目前只是个半成品，慢慢来吧。 
## 实现功能
- 支持http/https `html页面、restapi、文件流`
- 文件下载
- 自动分页并发爬取数据
- http代理设置


:heavy_exclamation_mark: :point_right: 本项目初衷纯属技术交流，勿作商业用途:heavy_exclamation_mark::bangbang:
